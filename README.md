# Boilerplate Project

## Overview

Welcome to the **Boilerplate Project**! This repository provides a foundational template for building React.js, Node.js applications.

### Features

- **Basic Setup:** Essential configuration for a Node.js, React.js application.
- **Modular Structure:** Organized and scalable project structure.
- **JWT Authentication:** User registration, login, password change, password reset email, and password reset functionalities located in the `jwtAuth` folder.
